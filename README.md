

How It Works
Scanning: A person places their RFID tag (ID card) near the RFID reader.

Processing: The microcontroller (like an Arduino or ESP32) reads the unique serial number of the tag.

Verification: The system checks if the ID exists in the database.

Logging: If the ID is valid, the system marks the person as "Present" and saves the timestamp

Software:
Arduino IDE: Used to write and upload the code to the microcontroller.

Programming Language: C++ (Arduino Sketch).

Libraries: MFRC522 (for the reader
